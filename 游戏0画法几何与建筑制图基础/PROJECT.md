# 游戏0：画法几何与建筑制图基础 - 项目说明

## 项目概览

**游戏名称**：画法几何与建筑制图基础  
**游戏编号**：游戏场0  
**主题**：画法几何与建筑制图入门  
**目标用户**：智能建造专业大一学生（手机端使用）  
**技术栈**：纯 HTML/CSS/JavaScript + Three.js

## 游戏定位

面向建筑学/智能建造专业大一新生的画法几何交互式学习系统，解决"空间想象难、投影理解难"的传统教学痛点。**手机端优先设计**，适配课堂随堂练习场景。

## 核心教学模块

### 1. 坐标系统模块

**功能特点**：
- ✅ 可切换坐标轴系统：
  - XY二维坐标系（平面几何）
  - XYZ三维坐标系（空间几何）
- 实时显示选中点的坐标值（精确到0.1）
- 支持两种创建点方式：
  - 点击空间任意位置创建点
  - 手动输入X/Y/Z坐标值精确定位
- 拖动点实时观察坐标变化

**关卡设计**：
- 关卡0-1：给坐标找点（输入坐标，在空间中定位）
- 关卡0-2：给点读坐标（点击空间点，读出坐标值）
- 关卡0-3：坐标连线（A、B两点坐标，观察空间直线）
- 关卡0-4：二维→三维坐标转换

---

### 2. 多视图联动模块

**视图系统**（共6个视图）：
- 平面图（俯视图）
- 正立面图（主视图）
- 左侧立面图
- 右侧立面图
- 背立面图
- **多种轴测图可切换**（核心需求）：
  - 正等轴测图（全等）
  - 正二等轴测图（二等）
  - 正面斜二测图（二斜）
  - 水平斜等测图（二水）

**核心交互**：
- 任意视图操作，其他视图同步更新
- 点击3D元素，各视图对应位置高亮
- 拖动3D点，观察五视图投影联动变化
- **一键切换轴测图类型**，对比观察差异

**关卡设计**：
- 关卡0-5：已知两点，画出第三面投影
- 关卡0-6：给定两面投影，求第三面投影
- 关卡0-7：根据五视图，还原3D空间位置
- 关卡0-8：四种轴测图对比识别

---

### 3. 平面图形与直线模块

**内容**：
- 投影面平行线：水平线、正平线、侧平线
- 投影面垂直线：铅垂线、正垂线、侧垂线
- 一般位置直线
- 实长求解：直角三角形法（动画演示）
- 平面的表示法：三点、两直线、平面图形

**交互设计**：
- 点击选择不同位置直线，观察投影特点
- 拖动改变直线位置，实时计算显示实长
- 直角三角形法步骤动画演示

---

### 4. 简单立体与切割模块

**立体类型**：
- 平面体：棱柱、棱锥、棱台
- 曲面体：圆柱、圆锥、球

**核心功能**（切割功能）：
- ✅ 平面切割立体
  - 水平切割面
  - 正垂切割面
  - 铅垂切割面
  - 一般位置切割面
- 实时显示截交线
- 切割后可分离观察两部分
- 切割面可拖动调整位置

**关卡设计**：
- 关卡0-9：平面切割棱柱，求截交线
- 关卡0-10：平面切割圆柱，求截交线
- 关卡0-11：平面切割圆锥，分析五种截交线类型
- 关卡0-12：根据切割后三视图，还原切割前立体

## 界面结构（手机端优先）

```
┌─────────────────┐
│   当前视图      │  ← 可切换：五视图/3D轴测
├─────────────────┤
│   工具按钮区    │  ← 选点/画线/切割/切换视图
├─────────────────┤
│   坐标输入区    │  ← X/Y/Z数值输入
├─────────────────┤
│   任务与提示    │  ← 当前关卡说明
└─────────────────┘
```

```
┌─────────┬─────────┐
│  视图1  │  视图2  │
├─────────┼─────────┤
│  视图3  │  视图4  │
└─────────┴─────────┘
         工具按钮
```

## 与游戏3的关系

**相同点**：
- 共享五视图系统架构
- 共享Three.js 3D渲染引擎
- 核心都是"空间↔平面"转换训练

**不同点**：
- 游戏0更基础，从点线面开始
- 游戏0强调坐标系统和精确输入
- 游戏0聚焦画法几何知识点
- 游戏0是手机端优先设计
- 游戏0包含立体切割功能
