# 游戏14隔墙与内装修

## 项目概述
建筑课程游戏化教学系统 - 隔墙与内装修

## 游戏信息
- **游戏编号**: 游戏14
- **游戏名称**: 隔墙与内装修
- **创建日期**: 2026年5月
- **教学目标**: 通过交互式游戏学习建筑设计相关知识

## 技术栈
- 前端: HTML5 + CSS3 + JavaScript
- 游戏引擎: 原生Canvas / DOM交互

## 飞书妙搭集成

- **集成日期**: 2026年5月7日
- **集成功能**: 添加飞书数据提交按钮
- **数据同步**: 游戏进度、关卡完成情况、得分自动同步到飞书多维表格
- **妙搭应用**: 建筑课程游戏化教学系统

### 同步字段
- 学号、姓名、班级
- 游戏编号、游戏名称
- 关卡编号、关卡名称、是否完成
- 得分、用时、操作数据JSON

## 功能特点
1. 交互式关卡设计
2. 实时进度保存
3. 飞书数据同步
4. 多端适配（PC/移动端

## 使用说明
1. 学生在飞书妙搭应用中打开游戏
2. 首次使用填写学号姓名
3. 完成关卡后点击"提交成绩到飞书"按钮
4. 数据自动同步到飞书多维表格
