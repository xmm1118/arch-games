# 建筑课程游戏化教学系统

> 面向建筑学专业的游戏化教学平台，包含 20 个互动游戏模块，覆盖从设计基础到构造深化的完整课程体系。

## 技术架构

- **前端**：纯 HTML + CSS + JavaScript（无构建依赖）
- **共享组件**：`shared-feishu-submission.js/css`、`shared-game-toolkit.js`、`shared-game-adapters.js`、`shared-draggable.js`、`shared-premium-theme.css`
- **部署**：GitHub Pages 静态托管
- **数据回传**：飞书多维表格（通过妙搭提交页）
