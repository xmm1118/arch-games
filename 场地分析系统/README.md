# 场地分析系统 v1.0

智能建造专业PBL教学游戏 - 房屋建筑学系列教育游戏之一

## 项目简介

这是一个基于Web的场地分析教学游戏，旨在帮助智能建造专业学生掌握场地分析的基本方法和技能。

## 游戏结构

### 六大关卡

| 关卡 | 名称 | 学习内容 |
|------|------|----------|
| 1 | 基础信息读取 | 场地选择、红线识别、地形识读 |
| 2 | 地形分析 | 坡度计算、朝向判断、高差利用 |
| 3 | 气候分析 | 热工分区识别、日照分析、风环境分析 |
| 4 | 周边环境 | 噪音分析、视线分析、景观资源评估 |
| 5 | 设计条件提取 | 退界测量、建筑限高、容积率计算 |
| 6 | 综合评估报告 | 数据汇总、设计建议、报告生成 |

### 内置场地案例

1. **城市中心地块** - 位于北京市朝阳区CBD核心区，商业用地
2. **郊区开发地块** - 位于上海市浦东新区临港新城，住宅用地
3. **山地开发地块** - 位于成都市龙泉驿区，坡地住宅
4. **沿街综合地块** - 位于广州市天河区，商住综合用地
5. **滨水景观地块** - 位于杭州市西湖区，滨水住宅

## 技术特点

- **单文件部署**：完整的HTML文件，无需服务器即可运行
- **Canvas地图交互**：支持等高线绘制、距离测量、多边形绘制
- **本地存储**：支持进度保存和恢复
- **响应式设计**：支持桌面和移动设备
- **PDF报告生成**：使用jsPDF库生成专业报告
- **JSON数据导出**：标准格式，支持后续设计系统读取

## 使用方法

### 直接运行

1. 双击 `index.html` 文件，在浏览器中打开
2. 或使用本地服务器：`python -m http.server 8000`

### 游戏流程

1. **选择场地**：从5个典型案例中选择一个
2. **完成分析**：按照关卡引导完成各项分析任务
3. **获取提示**：点击"💡 提示"按钮获取教学提示
4. **提交关卡**：每关完成后点击"提交"进入下一关
5. **生成报告**：完成所有关卡后生成PDF报告或JSON数据

## 数据输出

### PDF报告内容

- 场地基本信息
- 地形分析结果
- 气候分析数据
- 周边环境评估
- 设计条件汇总
- 综合设计建议

```json
  "design_suggestions": [...]
}
```

## 教学要点

### 关卡1：基础信息读取
- 建筑红线的概念和作用
- 等高线的识读方法
- 场地高差计算

### 关卡2：地形分析
- 坡度分级标准（<5°、5-15°、15-25°、>25°）
- 不同坡度的建筑适宜性
- 高差利用方案

### 关卡3：气候分析
- 中国热工分区标准（GB 50176）
- 日照分析基本方法
- 风环境与建筑布局

### 关卡4：周边环境
- 噪音源识别与防护
- 景观资源评估
- 日照遮挡分析

### 关卡5：设计条件提取
- 规划退界要求
- 容积率计算方法
- 建筑面积估算
